A propos

Un générateur de clés, aussi appelé keygen, est un programme informatique générant des numéros de série aléatoires afin d'activer un logiciel. Cette application vous permettra de générer des clés aléatoires, les enregistrer, afficher avec leurs dates et les supprimer.

Vous trouverez le code disponible sur https://github.com/nassim-fox/keygen/tree/master
